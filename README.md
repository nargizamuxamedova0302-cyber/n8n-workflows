# n8n Muvofiqlik Agenti (Compliance Agent)

Ushbu loyiha korporativ siyosatlar va me'yoriy hujjatlarni avtomatik tahlil qilish uchun n8n va Google Gemini AI asosida yaratilgan.

## Asosiy funksiyalari:

*   Matnni qabul qilish: Siyosat yoki me'yoriy hujjat matnlarini skanerlaydi.
*   Xatarlarni aniqlash: Kontent ichidagi muvofiqlik muammolari va xavf-xatarlarni tahlil qiladi.
*   Hisobot yaratish: Muvofiqlik bo'yicha batafsil hisobot va tavsiyalar tayyorlaydi.
*   Avtomatik ogohlantirish: Ixtiyoriy ravishda siyosatni yangilash yoki xavf haqida xabar berish jarayonini ishga tushiradi.

## Ishlatilgan vositalar:

*   n8n - Ish oqimini boshqarish (Workflow engine).
*   Google Gemini AI - Matnni chuqur tahlil qilish va xulosalar chiqarish uchun.
*   GitHub - Kod va ish jarayonini saqlash uchun.

## Qanday ishga tushiriladi:

1.  workflow.json faylini n8n-ga import qiling.
2.  Google Gemini API kalitini o'rnating.
3.  Ish jarayonini (workflow) faollashtiring.
